# My React Learning Journey 🚀

This repository contains my hands-on practice while learning React.
I am using this project to explore React fundamentals step by step with small examples and component-based experiments.

## What this project includes

- Function components
- Class components
- Props
- React Fragments
- Component organization in folders
- Basic Vite + React setup

## Project Structure

```bash
src/
	App.jsx
	Dummy.jsx
	main.jsx
	Components/
		ClassCompo.jsx
		FunCompo1.jsx
		FunCompo2.jsx
	props/
	ReactFragements/
		Fragment1.jsx
		Fragment2.jsx
```

## Getting Started

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
